# lisker-avatar-api
Lisker avatar API service, returns a lisker image based upon a given lisker Id.

### Example calls

Http calls are in the format:

`http://136.244.102.229:8030/liskerid/[id]?size=[size]`

For example:

[Female Lisker avatar size 1700](http://136.244.102.229:8030/liskerid/t111t311t414t491t522t532t547t551t701t801t856c1E4DAFDc2F84633c3AA5F38c42C1B18?size=1700)

[Male avatar size 400](http://136.244.102.229:8030/liskerid/t1128t313t416t492t522t535t544t552t701t801t855c1D9EADFc2FF830Cc3F8BB91c48D4731?size=600)
